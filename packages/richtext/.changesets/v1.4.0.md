- Added support to write custom HTML inside the following elements:

  - `<li>`
  - `<blockquote>`
  - `<h1>`
  - `<h2>`
  - `<h3>`
  - `<h4>`
  - `<h5>`
  - `<h5>`
  - `<h6>`

  Previously, only custom HTML inside `<p>` elements was parsed.
